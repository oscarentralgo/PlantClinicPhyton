# PlantClinic - Sistema de Diagnóstico IA para Plantas

## Descripción
PlantClinic es una aplicación web que utiliza IA para diagnosticar enfermedades y problemas en plantas de forma gratuita.

## Requisitos
- Python 3.7+
- Flask

## Instalación
1. Clona o descarga el proyecto
2. Crea un entorno virtual: `python -m venv venv`
3. Activa el entorno: `venv\Scripts\activate`
4. Instala dependencias: `pip install flask`

## Estructura del Proyecto
- `app.py` - Servidor Flask principal
- `faq.py` - Módulo de preguntas frecuentes
- `diagnósticos.py` - Módulo de diagnósticos
- `templates/` - Archivos HTML

## Rutas disponibles
- `/` - Página principal
- `/faq` - Preguntas frecuentes
- `/diagnosticos` - Diagnósticos de enfermedades
